static => 没有定位，top,left无效 默认属性不会脱离文档流
relative => 相对于自身正常位置向左，右，上，下移动 不影响其他元素的布局 不会脱离文档流
fixed => 相对于浏览器窗口，是固定位置，即使窗口是滚动的它也不会移动 脱离文档流
absolute如果元素没有已定位的父元素，那么它的位置 => 相对于html/body 脱离文档流
absolute => 相对于最近的已定位祖先元素：relative, absolute, fixed有效
